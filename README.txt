--------------------------------------------------------------------------------------------------------------------------------------

Button assignments:
    - Grip: The drop and the handle of the DPV can be gripped using the grip button.
    - Movement: When gripping the DPV with at least one controller, you can use the right trigger to adjust the DPV's speed.
    - Turning: If safe physical rotation is not guaranteed, the stick on the right controller can be used for horizontal rotation.
    - UI interaction: By pressing a button on the UI with the controller.
